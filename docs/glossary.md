# Glossary of Terms

This document provides definitions for technical and business terms used throughout the Globepay application and documentation.

## A

### ACH (Automated Clearing House)
A U.S. financial network that enables automatic money transfers between banks. Used for direct deposits, bill payments, and other electronic transfers.

### ACL (Access Control List)
A list of permissions attached to an object that specifies which users or system processes are granted access and what operations they can perform.

### ACID (Atomicity, Consistency, Isolation, Durability)
A set of properties that guarantee database transactions are processed reliably.

### ACL (Access Control List)
A list of permissions attached to an object that specifies which users or system processes are granted access and what operations they can perform.

### AES (Advanced Encryption Standard)
A symmetric encryption algorithm used to secure sensitive data.

### AML (Anti-Money Laundering)
Laws and regulations designed to prevent criminals from disguising illegally obtained funds as legitimate income.

### API (Application Programming Interface)
A set of protocols and tools for building software applications that specify how software components should interact.

### API Gateway
A management layer that sits between clients and backend services, providing features like authentication, rate limiting, and request/response transformation.

### ArgoCD
A declarative, GitOps continuous delivery tool for Kubernetes.

### Audit Trail
A chronological record of activities that provides documentary evidence of the sequence of activities affecting an operation, procedure, or event.

## B

### Backend
The server-side component of the application that handles business logic, data storage, and API endpoints.

### Bcrypt
A password hashing function designed to be slow and computationally expensive to prevent brute-force attacks.

### Beneficiary
A person or entity that receives money through a transfer.

### BGP (Border Gateway Protocol)
A standardized exterior gateway protocol used to exchange routing and reachability information between autonomous systems on the Internet.

### Blockchain
A distributed ledger technology that maintains a continuously growing list of records, called blocks, which are linked and secured using cryptography.

## C

### Cache
A hardware or software component that stores data so future requests for that data can be served faster.

### CAP Theorem
A concept that states it's impossible for a distributed data store to simultaneously provide more than two out of the following three guarantees: Consistency, Availability, and Partition tolerance.

### CDN (Content Delivery Network)
A geographically distributed network of proxy servers that cache content closer to users to improve load times.

### CI/CD (Continuous Integration/Continuous Deployment)
A method to frequently deliver apps to customers by introducing automation into the stages of app development.

### CloudFront
Amazon's content delivery network (CDN) service.

### Container
A lightweight, standalone, executable package that includes everything needed to run a piece of software.

### CORS (Cross-Origin Resource Sharing)
A mechanism that allows restricted resources on a web page to be requested from another domain outside the domain from which the first resource was served.

### CPU (Central Processing Unit)
The electronic circuitry within a computer that carries out the instructions of a computer program.

### CRUD (Create, Read, Update, Delete)
The four basic operations of persistent storage.

### CSRF (Cross-Site Request Forgery)
An attack that forces an end user to execute unwanted actions on a web application in which they're currently authenticated.

### CSP (Content Security Policy)
An added layer of security that helps to detect and mitigate certain types of attacks, including Cross-Site Scripting (XSS) and data injection attacks.

## D

### Database
An organized collection of structured information, or data, typically stored electronically in a computer system.

### Database Migration
The process of transferring data between different storage types, formats, or computer systems.

### DDoS (Distributed Denial of Service)
A malicious attempt to disrupt normal traffic of a targeted server, service, or network by overwhelming the target or its surrounding infrastructure with a flood of Internet traffic.

### Debit
An accounting entry that either increases an asset or expense account, or decreases a liability or equity account.

### Debit Card
A payment card that deducts money directly from a consumer's bank account to pay for a purchase.

### DevOps
A set of practices that combines software development and IT operations to shorten the systems development life cycle and provide continuous delivery with high software quality.

### Digital Wallet
An electronic device or online service that allows individuals to make electronic transactions.

### Docker
A platform for developing, shipping, and running applications in containers.

### Docker Compose
A tool for defining and running multi-container Docker applications.

### Domain
A distinct area of activity, interest, or knowledge, or a particular sphere of activity or function.

## E

### EBS (Elastic Block Store)
Amazon's block storage service for EC2 instances.

### ECS (Elastic Container Service)
Amazon's container orchestration service.

### EKS (Elastic Kubernetes Service)
Amazon's managed Kubernetes service.

### ETL (Extract, Transform, Load)
The process of extracting data from various sources, transforming it to fit business needs, and loading it into a target database or data warehouse.

### E-Wallet
An electronic device or online service that allows individuals to make electronic transactions.

### Exchange Rate
The rate at which one currency can be exchanged for another.

## F

### Fargate
AWS's serverless compute engine for containers.

### FCM (Firebase Cloud Messaging)
A cross-platform messaging solution that lets you reliably deliver messages at no cost.

### FIFO (First In, First Out)
A method for organizing and manipulating a data buffer where the oldest entry is processed first.

### Firewall
A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.

### Frontend
The client-side component of the application that users interact with directly.

## G

### GDPR (General Data Protection Regulation)
A regulation in EU law on data protection and privacy for individuals within the European Union and the European Economic Area.

### Git
A distributed version control system for tracking changes in source code during software development.

### GitHub Actions
A CI/CD platform that allows you to automate your software development workflows.

### Go (Golang)
An open-source programming language developed by Google.

### Grafana
An open-source platform for monitoring and observability.

### GraphQL
A query language for APIs and a runtime for fulfilling those queries with existing data.

## H

### HA (High Availability)
A characteristic of a system that aims to ensure an agreed level of operational performance, usually uptime, for a higher than normal period.

### Helm
A package manager for Kubernetes that makes it easy to find, share, and use software built for Kubernetes.

### HMAC (Hash-based Message Authentication Code)
A specific type of message authentication code involving a cryptographic hash function and a secret cryptographic key.

### Horizontal Scaling
Adding more machines or nodes to a pool of resources rather than increasing the power of a single machine.

### HTTP (Hypertext Transfer Protocol)
The foundation of data communication for the World Wide Web.

### HTTPS (HTTP Secure)
An extension of HTTP used for secure communication over a computer network.

## I

### IAM (Identity and Access Management)
A framework of policies and technologies to ensure that the proper people in an enterprise have the appropriate access to technology resources.

### IDE (Integrated Development Environment)
A software application that provides comprehensive facilities to computer programmers for software development.

### Idempotent
An operation that has the same effect whether it is performed once or multiple times.

### Index (Database)
A data structure that improves the speed of data retrieval operations on a database table.

### Infrastructure as Code (IaC)
The process of managing and provisioning computer data centers through machine-readable definition files.

### Ingress
Kubernetes API object that manages external access to the services in a cluster, typically HTTP.

### Integration Testing
A level of software testing where individual units are combined and tested as a group.

### IoT (Internet of Things)
The network of physical objects embedded with sensors, software, and other technologies for the purpose of connecting and exchanging data with other devices and systems over the internet.

## J

### Jaeger
An open-source distributed tracing system.

### JavaScript
A programming language that conforms to the ECMAScript specification.

### JWT (JSON Web Token)
An open standard for creating access tokens that assert some number of claims.

## K

### K8s (Kubernetes)
An open-source container orchestration system for automating software deployment, scaling, and management.

### KMS (Key Management Service)
A managed service that makes it easy for you to create and control the encryption keys used to encrypt your data.

### KPI (Key Performance Indicator)
A type of performance measurement that evaluates the success of an organization or a particular activity.

### Kubeconfig
A configuration file that defines cluster access parameters for kubectl.

### Kubernetes
An open-source container orchestration system for automating software deployment, scaling, and management.

### KYC (Know Your Customer)
The process of a business identifying and verifying the identity of its clients.

## L

### Latency
The time delay between the cause and the effect of some physical change in the system being used.

### LDAP (Lightweight Directory Access Protocol)
An open, vendor-neutral, industry standard application protocol for accessing and maintaining distributed directory information services over an IP network.

### Load Balancer
A device that acts as a reverse proxy and distributes network or application traffic across a number of servers.

### Loki
A horizontally-scalable, highly-available, multi-tenant log aggregation system inspired by Prometheus.

### Logging
The practice of recording events, transactions, and system activities for debugging, monitoring, and auditing purposes.

## M

### Microservices
An architectural style that structures an application as a collection of loosely coupled services.

### Middleware
Software that provides common services and capabilities to applications outside of what's offered by the operating system.

### Migration
The process of moving data, applications, or other business elements from one environment to another.

### MFA (Multi-Factor Authentication)
A method of computer access control in which a user is only granted access after successfully presenting several separate pieces of evidence to an authentication mechanism.

### Monitoring
The process of collecting, analyzing, and using information to track the performance of applications and infrastructure.

### MQ (Message Queue)
A form of asynchronous service-to-service communication used in serverless and microservices architectures.

## N

### NAT (Network Address Translation)
A method of remapping one IP address space into another by modifying network address information in the IP header of packets while they are in transit.

### Node
A worker machine in Kubernetes that runs pods.

### npm (Node Package Manager)
The default package manager for Node.js.

### NTP (Network Time Protocol)
A networking protocol for clock synchronization between computer systems.

## O

### OAuth
An open standard for access delegation commonly used as a way for Internet users to grant websites or applications access to their information on other websites but without giving them the passwords.

### Observability
The ability to measure the internal states of a system by examining its outputs.

### OIDC (OpenID Connect)
An authentication layer on top of OAuth 2.0, an open standard for access delegation.

### ORM (Object-Relational Mapping)
A programming technique for converting data between incompatible type systems using object-oriented programming languages.

## P

### P95/P99
Percentile metrics that measure the 95th and 99th percentiles of response times or other performance metrics.

### PCI DSS (Payment Card Industry Data Security Standard)
A set of security standards designed to ensure that all companies that accept, process, store, or transmit credit card information maintain a secure environment.

### Pod
The smallest deployable units of computing that you can create and manage in Kubernetes.

### PostgreSQL
An open-source relational database management system.

### Prometheus
A systems monitoring and alerting toolkit originally built at SoundCloud.

### Promtail
An agent that ships the contents of local logs to a Loki instance.

### Proxy
An intermediary server that separates clients from the destination servers they request resources from.

## Q

### Quorum
The minimum number of nodes that must be available for a distributed system to function properly.

## R

### RDS (Relational Database Service)
Amazon's managed relational database service.

### RBAC (Role-Based Access Control)
A method of regulating access to computer or network resources based on the roles of individual users within an enterprise.

### React
A JavaScript library for building user interfaces.

### Redis
An open-source, in-memory data structure store, used as a database, cache, and message broker.

### Redux
A predictable state container for JavaScript apps.

### Refund
The return of money paid for a product or service.

### REST (Representational State Transfer)
An architectural style for designing networked applications.

### RPO (Recovery Point Objective)
The maximum target period in which data might be lost from an IT service due to a major incident.

### RTO (Recovery Time Objective)
The targeted duration of time and a service level within which a business process must be restored after a disaster.

### RSA
A public-key cryptosystem widely used for secure data transmission.

## S

### S3 (Simple Storage Service)
Amazon's object storage service.

### SaaS (Software as a Service)
A software licensing and delivery model in which software is licensed on a subscription basis and is centrally hosted.

### Salt
A random data used as an additional input to a one-way function that hashes data, passwords, or passphrases.

### Schema
The structure of a database system, described in a formal language supported by the database management system.

### SDK (Software Development Kit)
A collection of software development tools in one installable package.

### Secret
An object that contains a small amount of sensitive data such as a password, a token, or a key.

### Security Group
A virtual firewall that controls inbound and outbound traffic for AWS resources.

### SEO (Search Engine Optimization)
The process of increasing the quality and quantity of website traffic to a website or a web page from search engines.

### SES (Simple Email Service)
Amazon's cloud-based email sending service.

### SLA (Service Level Agreement)
A commitment between a service provider and a client that specifies the performance standards the provider must meet.

### SMS (Short Message Service)
A text messaging service component of most telephone, internet, and mobile device systems.

### SMTP (Simple Mail Transfer Protocol)
A communication protocol for email transmission.

### SNMP (Simple Network Management Protocol)
An Internet Standard protocol for collecting and organizing information about managed devices on IP networks.

### SOA (Service-Oriented Architecture)
A style of software design where services are provided to other components by application components, through a communication protocol over a network.

### SOAP (Simple Object Access Protocol)
A messaging protocol specification for exchanging structured information in the implementation of web services.

### SOC 2 (Service Organization Control 2)
An auditing procedure that ensures service providers securely manage data to protect the interests of their clients.

### SPA (Single Page Application)
A web application or website that interacts with the user by dynamically rewriting the current page rather than loading entire new pages from a server.

### SQL (Structured Query Language)
A domain-specific language used in programming and designed for managing data held in a relational database management system.

### SQL Injection
A code injection technique that might destroy your database.

### SQS (Simple Queue Service)
Amazon's fully managed message queuing service.

### SSL (Secure Sockets Layer)
A standard security technology for establishing an encrypted link between a server and a client.

### SSH (Secure Shell)
A cryptographic network protocol for operating network services securely over an unsecured network.

### SSO (Single Sign-On)
An authentication scheme that allows a user to log in with a single ID and password to any of several related, yet independent, software systems.

### Stateful
A system that stores information about previous interactions or events.

### Stateless
A system that does not store any information about previous interactions or events.

### Swagger
An open-source software framework backed by a large ecosystem of tools that helps developers design, build, document, and consume RESTful web services.

## T

### TDD (Test-Driven Development)
A software development process that relies on the repetition of a very short development cycle: requirements are turned into very specific test cases.

### Terraform
An open-source infrastructure as code tool that lets you provision cloud resources with declarative configuration files.

### TLS (Transport Layer Security)
A cryptographic protocol that provides communications security over a computer network.

### TOTP (Time-Based One-Time Password)
An algorithm that computes a one-time password from a shared secret key and the current time.

### Transaction
A sequence of database operations that are treated as a single unit of work.

### TypeScript
A programming language that builds on JavaScript by adding static type definitions.

## U

### UI (User Interface)
The space where interactions between humans and machines occur.

### Unit Testing
A software testing method by which individual units of source code are tested to determine whether they are fit for use.

### Unix Timestamp
A way of representing a point in time as the number of seconds that have elapsed since January 1, 1970.

### URL (Uniform Resource Locator)
A reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.

### UX (User Experience)
A person's emotions and attitudes about using a particular product, system, or service.

## V

### VPC (Virtual Private Cloud)
A virtual network dedicated to your AWS account.

### VPN (Virtual Private Network)
A service that encrypts your internet traffic and disguises your online identity.

## W

### WAF (Web Application Firewall)
A specific form of application firewall that filters, monitors, and blocks HTTP traffic to and from a web service.

### WebSocket
A computer communications protocol, providing full-duplex communication channels over a single TCP connection.

### Webhook
A way for an app to provide other applications with real-time information through HTTP callbacks.

## X

### XSS (Cross-Site Scripting)
A type of security vulnerability typically found in web applications that enables attackers to inject client-side scripts into web pages viewed by other users.

## Y

### YAML (YAML Ain't Markup Language)
A human-readable data serialization standard that can be used in conjunction with all programming languages.

## Z

### Zero Trust
A security concept centered on the belief that organizations should not automatically trust anything inside or outside its perimeters.

This glossary provides definitions for terms commonly used in the Globepay application and its documentation. If you encounter a term not defined here, please consult the relevant technical documentation or reach out to the development team.
